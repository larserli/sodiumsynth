Detailed Description

Trigger based on rolling average

This module maintains a rolling average over a window. If the mean value exceeds a treshold, an action is raised, if defined. Another action is raised when the mean drops below the threshold. It is possible to define a hysteresis to avoid repeated actions when the level is close to the treshold.
